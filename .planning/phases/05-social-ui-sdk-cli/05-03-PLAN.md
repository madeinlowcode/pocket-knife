Funcao main():
1. Exibir banner: chalk.bold.blue('\npocket-knife init') + '\nConfigure your AI provider keys in ~/.claude/.env\n'
2. Verificar Node.js version: if (parseInt(process.versions.node) < 18) exibir erro e exit 1
3. Ler chaves existentes: const envPath = getEnvPath(); const existingKeys = existsSync(envPath) ? dotenv.parse(fs.readFileSync(envPath, 'utf-8')) : {};
4. Selecionar categorias: const selectedIds = await selectCategories();
5. Se nenhuma categoria selecionada: chalk.yellow('\nNo categories selected. Run again to configure.') e exit 0
6. Coletar chaves: const newKeys = await promptKeys(selectedIds, existingKeys);
7. Se nenhuma chave nova: chalk.green('\nAll selected keys are already configured!') e exit 0
8. Validar cada chave nova:
   Para cada [envVar, key] em Object.entries(newKeys):
     process.stdout.write('  Validating ' + envVar + '... ')
     const result = await validateKey(envVar, key)
     if (result.skipped) console.log(chalk.gray('skipped (no validation endpoint)'))
     else if (result.valid) console.log(chalk.green('valid'))
     else:
       console.log(chalk.yellow('could not validate: ' + (result.error || 'check key')))
       const save = await confirm({ message: 'Save anyway?', default: false })
       if (!save) delete newKeys[envVar]
9. Escrever chaves no .env: const { added, skipped } = await mergeEnv(newKeys)
10. Exibir sumario: chalk.bold.green('\nSetup complete!') + '\n  Added: ' + added.join(', ') + '\n  Already set: ' + skipped.join(', ')
11. Exibir proximo passo: '\nRestart your Claude Code session for the keys to take effect.\nRun /pocket-knife:setup inside Claude Code to verify.\n'

main().catch(err => { console.error(chalk.red('Error: ' + err.message)); process.exit(1); });
