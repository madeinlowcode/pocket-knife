Shebang obrigatorio na linha 1: #!/usr/bin/env node
Usar APENAS imports de built-ins Node.js: node:crypto e node:https (ZERO dependencias npm)
Implementar OAuth 1.0a HMAC-SHA1 conforme spec oficial X Developer docs.

Estrutura do helper:
1. Importar: import { createHmac, randomBytes } from 'node:crypto'; import { request } from 'node:https';
2. Ler env vars: X_CONSUMER_KEY, X_CONSUMER_SECRET, X_ACCESS_TOKEN, X_ACCESS_TOKEN_SECRET, ACTION, TWEET_TEXT, TWEET_ID, USER_ID
3. Validar que as 4 credenciais existem; se nao, console.error + process.exit(1)
4. funcao percentEncode(str): encodeURIComponent(String(str)).replace(/[!'()*]/g, c => '%' + c.charCodeAt(0).toString(16).toUpperCase())
5. funcao buildOAuthHeader(method, url, bodyParams={}):
   - timestamp = Math.floor(Date.now() / 1000).toString()
   - nonce = randomBytes(16).toString('hex')
   - oauthParams = { oauth_consumer_key, oauth_nonce, oauth_signature_method: 'HMAC-SHA1', oauth_timestamp, oauth_token, oauth_version: '1.0' }
   - allParams = { ...oauthParams, ...bodyParams }
   - sortedParams = Object.keys(allParams).sort().map(k => percentEncode(k) + '=' + percentEncode(allParams[k])).join('&')
   - baseString = METHOD.toUpperCase() + '&' + percentEncode(url) + '&' + percentEncode(sortedParams)
   - signingKey = percentEncode(consumerSecret) + '&' + percentEncode(accessTokenSecret)
   - signature = createHmac('sha1', signingKey).update(baseString).digest('base64')
   - retornar 'OAuth ' + Object.keys({...oauthParams, oauth_signature: signature}).sort().map(k => k + '="' + percentEncode(v) + '"').join(', ')
6. funcao httpsRequest(method, hostname, path, headers, body=null): retorna Promise com { status, body }
7. funcao main() async com switch(action):
   case 'post':   POST https://api.twitter.com/2/tweets com body {"text": tweetText}, OAuth header sem body params na assinatura
   case 'like':   POST https://api.twitter.com/2/users/${userId}/likes com body {"tweet_id": tweetId}
   case 'retweet':POST https://api.twitter.com/2/users/${userId}/retweets com body {"tweet_id": tweetId}
   case 'delete': DELETE https://api.twitter.com/2/tweets/${tweetId} sem body
   case 'me':     GET https://api.twitter.com/2/users/me sem body
   default: console.error + process.exit(1)
8. main().catch(err => { console.error('ERROR:', err.message); process.exit(1); });
