# Design Guide

## Logo Design

**Process:**
1. Define brand personality (3 adjectives: e.g., "bold, modern, trustworthy")
2. Choose style: wordmark, lettermark, icon+text, abstract icon
3. Select color palette (2-3 colors max for logos)
4. Test at small sizes (16px favicon) and large sizes

**Color psychology:**
| Color | Associations | Industries |
|-------|-------------|------------|
| Blue | Trust, stability | Finance, tech, healthcare |
| Green | Growth, nature | Sustainability, health, finance |
| Red | Energy, urgency | Food, retail, entertainment |
| Black | Luxury, sophistication | Fashion, premium products |
| Orange | Creativity, warmth | Food, startups, media |

**Logo prompt template for AI:**
`"Minimalist [STYLE] logo for [BRAND], [COLOR1] and [COLOR2] color palette, clean lines, vector style, white background, professional"`

## OG Image Design (Social Share)

**Specs:** 1200x630px, under 8MB, JPG or PNG

**Layout formula:**
```
[BACKGROUND: brand color or image]
[LOGO: top-left, max 20% width]
[HEADLINE: center, 60-80px, high contrast]
[SUBHEADLINE: below headline, 30-40px]
[CTA or URL: bottom, smaller]
```

**Common mistakes:**
- Text too close to edges (safe zone: 60px from each edge)
- Low contrast (test with grayscale)
- Too much text (max 6-8 words for headline)
- Wrong aspect ratio (platforms crop differently)

## Landing Page Design

**Above-the-fold checklist:**
- [ ] Clear headline stating the value proposition (what, for whom)
- [ ] Subheadline with supporting detail (how, proof point)
- [ ] Primary CTA button (high contrast, action verb: "Start Free", "Get Demo")
- [ ] Hero image or product screenshot
- [ ] Social proof (logos, numbers, testimonial snippet)

**Section order:**
1. Hero (value prop + CTA)
2. Problem (pain point)
3. Solution (product/service)
4. Social proof (testimonials, logos, metrics)
5. Features (how it works)
6. Pricing (if applicable)
7. FAQ
8. Final CTA

## YouTube Thumbnail Design

**Formula:** Emotion face + text overlay + bold colors

| Element | Guideline |
|---------|-----------|
| Face | Large, expressive, 60%+ of frame |
| Text | Max 3-5 words, 80px+, high contrast |
| Colors | 2-3 bold colors, avoid muted tones |
| Contrast | Test in small size (90x54px) |
| Consistency | Same style across channel for brand recognition |

**Prompt template:**
`"YouTube thumbnail, [SUBJECT] with [EMOTION] expression, bold [COLOR] text overlay '[TITLE]', high contrast, eye-catching, professional"`

## App Store Screenshots

**Strategy:** Tell a story in 3 screenshots
1. Screenshot 1: Core value proposition headline + key feature
2. Screenshot 2: Second most important feature
3. Screenshot 3: Social proof or unique differentiator

**Text overlays:** Use device frames, keep text under 5 words per screenshot, use action-oriented captions
