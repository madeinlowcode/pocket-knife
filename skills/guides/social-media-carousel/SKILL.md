# Social Media Carousel Design

## What Makes a Carousel Work

Carousels succeed when each slide creates a micro-reason to swipe. The goal is to manufacture curiosity at every slide edge. A poorly structured carousel loses 60–80% of viewers by slide 3.

---

## Platform Dimensions

| Platform | Recommended Size | Aspect Ratio | File Format | Max Slides |
|----------|-----------------|--------------|-------------|------------|
| Instagram | 1080 x 1080 px | 1:1 (square) | JPG/PNG | 10 |
| Instagram (portrait) | 1080 x 1350 px | 4:5 | JPG/PNG | 10 |
| LinkedIn (PDF) | 1080 x 1080 px | 1:1 | PDF | 300 |
| LinkedIn (portrait) | 1080 x 1350 px | 4:5 | PDF | 300 |
| Facebook | 1080 x 1080 px | 1:1 | JPG/PNG | 10 |

> LinkedIn carousels are uploaded as PDFs. Each page becomes one slide. Use portrait format for maximum screen real estate on mobile.

## The 10-Slide Carousel Structure

### Slide 1 — Cover (Hook)
**Job:** Stop the scroll. Earn the first swipe.
- Bold, high-contrast headline
- Clear visual hierarchy: one dominant element
- No more than 7 words in the main headline
- Include a "swipe to learn" or visual arrow indicator
- Must work as a standalone image (shown in feed before swiping)

### Slide 2 — Problem or Promise
**Job:** Confirm why this carousel is worth their time.
- State the core tension: what problem does this solve?
- Or restate the promise more specifically than slide 1
- Keep copy under 20 words

### Slides 3–8 — Content Body
**Job:** Deliver value in scannable, self-contained units.
- One idea per slide (never split a thought across two slides)
- Use consistent layout grid (don't redesign from slide to slide)
- Alternate between text-heavy and visual slides to create rhythm
- Add a slide number (3/8) for progress signaling
- Each slide should hint at the next ("But that's not the hardest part...")

### Slide 9 — Summary or Key Takeaway
**Job:** Reinforce the most important lesson.
- Repeat or synthesize the core message
- "If you remember only one thing from this..."
- Visually distinct from body slides (use accent color or layout shift)

### Slide 10 — CTA Slide
**Job:** Convert attention into action.
- One CTA only (follow, save, comment, DM, visit link)
- Reinforce brand identity: logo, handle, brand color
- Optional: lead magnet hook ("Save this for later")

## Visual Consistency Rules

### Typography
- Maximum **2 typefaces** per carousel (one for headers, one for body)
- Header size: 48–72 pt | Body size: 20–28 pt
- Line spacing: 1.3–1.5x for readability on mobile
- Never use light gray text on white background

### Color
- Define a 3-color palette before starting: primary, accent, neutral
- Use the accent color for emphasis only (1–2 elements per slide)
- Maintain consistent background color across body slides
- CTA slide can break palette as a visual interrupt

### Layout Grid
- Use a consistent safe zone: 80–100 px margins on all sides
- Align all elements to the same grid across slides
- Avoid elements bleeding off the edge (crop risk on some platforms)

### Imagery
- Use consistent image style: all illustrations, all photos, or all icons
- Do not mix photographic and illustrated styles mid-carousel
- Image treatment: consistent filter, saturation, or overlay across all slides

## Swipe Hooks (Between-Slide Cliffhangers)

Add a micro-hook at the bottom or edge of each body slide:

```
"The next one surprises most people →"
"Slide 5 is where most get stuck →"
"You probably do this wrong too →"
"Wait until you see #7 →"
"This changed everything for me →"
```

These can be small text at the bottom of the slide or embedded in the visual layout.

---

## CTA Placement Strategy

| CTA Type | Best Placement |
|----------|---------------|
| Follow | Slide 1 (bottom) + Slide 10 |
| Save | Slide 2 and Slide 10 |
| Comment with answer | Slide where you pose a question |
| DM for resource | Slide 10 |
| Link in bio | Slide 10 only |

Avoid putting links on mid-carousel slides — viewers rarely stop swiping to click.

## Carousel Script Template (AI Prompt)

```
Create a 10-slide carousel about [topic] for [Instagram / LinkedIn].

Audience: [describe target audience]
Goal: [educate / generate leads / build authority / drive saves]

Output format per slide:
- Slide number
- Headline (under 8 words)
- Body copy (under 30 words)
- Visual suggestion
- Swipe hook (for slides 2–9)

Slide 1 must be a bold hook. Slide 10 must have a single CTA.
```

## Common Carousel Mistakes

- **Too much text per slide** — Viewers scan, they don't read. Max 30 words per slide.
- **Inconsistent layout** — Each slide feels like a different designer made it.
- **No swipe incentive** — Slides 2–3 don't tease what's coming.
- **Weak cover slide** — The cover must compete with Reels and viral photos in the same feed.
- **CTA on every slide** — One CTA, on the last slide. More than that = confusion.
- **No brand identity** — Someone screenshots slide 4 and shares it. Can anyone find you?

---

## Design Checklist

- [ ] Cover slide has a bold hook under 7 words
- [ ] Platform dimensions correct (square or 4:5)
- [ ] Consistent 2-typeface system across all slides
- [ ] 3-color palette applied uniformly
- [ ] 80–100 px safe zone margins on all slides
- [ ] Slide progress indicator present (e.g., 3/10)
- [ ] Swipe hook text on each body slide
- [ ] One CTA on the final slide only
- [ ] Brand handle/logo on cover and CTA slide
- [ ] Exported at correct resolution (72 dpi screen, 150 dpi for print-quality PDF)
- [ ] Tested on mobile screen at actual size before publishing
