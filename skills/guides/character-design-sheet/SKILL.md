# Character Design Sheet Guide

## Core Purpose

A character design sheet (also called a model sheet or character bible page) is the definitive reference document for a character. It ensures visual consistency across all media, artists, and production stages. Every element must be reproducible from this document alone.

---

## What a Complete Design Sheet Contains

### Minimum Viable Sheet
- Front-facing orthographic view (full body)
- Side profile view
- Back view
- Character height reference
- Signature color palette

### Professional Production Sheet
- Full turnaround (front, 3/4, side, back, 3/4 rear)
- Expression sheet (6–12 expressions)
- Hand/detail studies
- Accessory/costume details
- Color palette with hex/Pantone/CMYK codes
- Proportion guide with measurements
- Typography or logo treatment (for branding characters)

---

## Turnaround Views

### The Five-Point Turnaround
Position the character on a consistent horizon line across all views:

1. **Front (0°)**: Primary design view. All symmetry visible.
2. **Three-quarter front (45°)**: Most dynamic, shows depth and form
3. **Side (90°)**: Silhouette test — readable in complete profile
4. **Three-quarter rear (135°)**: Reveals back design, hair, costume details
5. **Back (180°)**: Back costume, hair, and rear silhouette

### Turnaround Technical Rules
- All views on the same baseline (feet touch the same horizontal line)
- All views at the same scale
- Eyeline consistent across all views
- Shoulder width, hip width, and head size identical in equivalent views
- Use perspective guidelines if showing 3/4 views

### Common Turnaround Mistakes
- Head size inconsistency between views (most common error)
- Feet not on shared baseline
- Hair volume changes between views without justification
- Costume details that are physically impossible across views

## Expression Sheets

### Core 6 Expressions (minimum)
1. **Neutral** — resting state, base reference for all others
2. **Happy** — genuine smile, engaged eyes
3. **Angry** — the character's anger, not generic rage
4. **Sad** — their specific vulnerability
5. **Surprised** — reaction intensity appropriate to personality
6. **Determined / Focused** — their "hero moment" expression

### Extended Expression Set (12 expressions)
Add to core 6:
7. **Disgusted**
8. **Fearful / Scared**
9. **Laughing / Delighted**
10. **Confused / Skeptical**
11. **Shy / Embarrassed**
12. **Smug / Confident** (personality-specific)

### Expression Sheet Guidelines
- Heads should be at identical scale across the sheet
- Ears and overall head shape remain constant — only features move
- Eyes, brows, and mouth are the primary expression vehicles
- Avoid extremes that distort the head shape (unless that's the style)
- Each expression should feel like THIS character, not a generic face

### Eyebrow and Eye Movement Reference
- Brows move independently of each other for nuanced reads
- Inner corners raise = sadness/worry
- Inner corners lower = anger
- Outer corners raise = happiness
- Wide eyes = surprise/fear
- Squinted eyes = anger/suspicion/focus

## Proportion Guides

### Head-Based Measurement System
Use the character's own head height as the unit of measure.

Common proportions:
- **Realistic adult**: 7–8 heads tall
- **Heroic/Action**: 8–9 heads tall (elongated, powerful)
- **Stylized/Cartoon**: 4–6 heads tall
- **Super-deformed (chibi)**: 2–3 heads tall

### Proportion Annotation
On the front view, draw horizontal guidelines marking:
- Top of head
- Chin
- Shoulder line
- Chest/bust line
- Waist
- Hip line
- Crotch/mid-body
- Knee line
- Ankle line

Label each with its head-unit measurement (e.g., "Waist = 3.5 heads from top")

### Width Proportions
- Shoulder width: typically 2–2.5 head widths for adults
- Hip width: typically 1.5–2 head widths
- Note any asymmetry intentionally in the sheet

## Color Palette

### Palette Structure
Each character needs a complete, named color system:

- **Base colors**: Primary skin, hair, costume colors (4–6 colors)
- **Shadow colors**: Pre-defined darks for each base (1 shadow per base)
- **Highlight colors**: Pre-defined lights or specular hits (1–2 per base)
- **Accent colors**: Small but important details (pupils, accessories)

### Color Documentation Format
For each color, document:
- Hex code (for digital): `#F2C89B`
- RGB values: `R:242 G:200 B:155`
- Pantone (for print): `Pantone 7507 C`
- CMYK (for print): `C:0 M:17 Y:36 K:5`
- Descriptive name: "Warm Peach Skin Base"

### Color Harmony Rules
- Characters in the same universe should share at least one color
- Primary characters: 3–4 base colors maximum (readable silhouette)
- Secondary characters: Can have busier palettes
- Villain vs. hero: Often use opposing temperature colors (warm hero, cool villain)

### Lighting Reference Swatches
Include a small sphere or cylinder rendered in:
- Flat color (no lighting)
- With standard 3/4 top-left light
- With ambient/environmental light variant
This helps artists apply shading consistently.

## Style Consistency Framework

### Style Anchors
Define the stylistic rules that apply to ALL characters in this universe:

- **Line weight**: Uniform? Varied? Tapered? What weight range?
- **Face simplification level**: How many lines describe a nose? An ear?
- **Rendering approach**: Flat, cel-shaded, painterly, textured?
- **Eye style**: Anime-influenced, western cartoon, realistic?
- **Anatomy stylization**: Exaggerated which direction (long limbs? big head?)

### Cross-Character Consistency Tests
When designing multiple characters, verify:
- They look like they exist in the same world
- Scale relationships are correct when placed side-by-side
- Line weight approach is identical
- Level of detail is consistent

### Silhouette Test
Every character should be immediately recognizable as a solid black silhouette. Test this on your design:
- Cut out the silhouette shape
- Can you tell who it is without any internal detail?
- Is the silhouette unique from other characters in the cast?

## Special Elements

### Accessories and Props
Dedicated studies for:
- Weapons (multiple angles, grip position)
- Bags, armor, pouches (open and closed states)
- Signature objects (a character's hat, cane, specific tool)

### Costume Variations
On a single sheet or supplemental sheets:
- Casual / alternate outfit
- Formal / special occasion
- Action / equipped state
- Seasonal variants (if applicable)

### Age Variations (if applicable)
For characters shown at different ages:
- Proportion shifts per age (children have larger heads relative to body)
- Consistent identity markers (birthmark, eye color, hair shape)
- Height chart showing growth stages

---

## Sheet Layout Best Practices

### Arrangement
- Front turnaround: center or left-center, largest element
- Profile and back views: same row, same scale
- Expressions: grouped grid, below or beside turnarounds
- Color palette: bottom or right margin, clearly labeled
- Notes/callouts: use leader lines, never obscure the art

### File Organization
- Background: white or neutral gray (never pattern)
- 300 DPI minimum for print
- RGB for digital distribution, CMYK for print
- Layer organization: Linework / Flats / Shading / Annotations / Background

### Title Block
Include in corner of every sheet:
- Character name
- Project/property name
- Version number (v1.0, v2.1)
- Artist name
- Date
