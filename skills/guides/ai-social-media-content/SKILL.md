# AI Social Media Content Creation

## Platform Quick Reference

| Platform | Ideal Length | Best Format | Posting Frequency |
|----------|-------------|-------------|-------------------|
| Instagram | 138–150 chars (caption) | Carousel, Reel | 4–7x/week |
| LinkedIn | 900–1200 chars | Text + image, Carousel | 3–5x/week |
| Twitter/X | 240–280 chars | Thread, single | 1–3x/day |
| TikTok | 150 chars (caption) | Vertical video | 1–3x/day |
| Facebook | 40–80 chars | Video, image | 1–2x/day |
| YouTube Shorts | 100 chars (title) | Vertical video | 3–5x/week |

---

## Content Pillars Framework

Define 3–5 recurring content pillars before generating anything:

1. **Educational** — How-to, explainers, tips, frameworks
2. **Inspirational** — Stories, transformations, quotes in context
3. **Promotional** — Product features, offers, case studies
4. **Engagement** — Questions, polls, controversial opinions
5. **Behind the Scenes** — Process, team, building in public

> Rule: No more than 20% promotional content. The 80/20 mix keeps audiences engaged without feeling sold to.

## AI Prompting Frameworks by Platform

### Instagram

```
Write an Instagram caption for [topic].
- Hook in the first line (no hashtags in line 1)
- 3–5 short paragraphs with line breaks
- Call to action in the final line
- 15–20 hashtags in first comment (not caption)
- Tone: [conversational/educational/inspirational]
```

### LinkedIn

```
Write a LinkedIn post about [topic] using this structure:
- Line 1: Bold hook (no "I am excited to announce")
- Lines 2–5: Context or story setup
- Lines 6–10: Key insight or lesson
- Final 2 lines: Actionable takeaway + question for comments
- 3–5 relevant hashtags at the end
- Target reader: [job title / audience]
```

### Twitter/X

```
Write a 6-tweet thread about [topic].
Tweet 1: Hook with bold claim or surprising stat
Tweets 2–5: One key point each, no padding
Tweet 6: Summary + CTA (follow, retweet, reply)
Each tweet under 280 characters. Number as 1/ 2/ etc.
```

### TikTok / Reels Script

```
Write a 45-second script for [topic] vertical video.
- Second 0–3: Pattern interrupt hook (visual + spoken)
- Second 4–20: Main value delivery (3 quick points)
- Second 21–40: Proof or example
- Second 41–45: CTA (follow, save, comment)
Include on-screen text suggestions in [brackets].
```

## Hashtag Strategy

### Tiers (Instagram Model)

- **Tier 1 — Niche (3–5 tags):** Under 100K posts. High chance of ranking.
- **Tier 2 — Mid (5–8 tags):** 100K–1M posts. Balanced reach.
- **Tier 3 — Broad (3–5 tags):** 1M+ posts. Visibility play.
- **Branded (1–2 tags):** Your own hashtag. Always include.

### AI Prompt for Hashtag Research

```
Generate 20 Instagram hashtags for a post about [topic] in the [niche] space.
Sort into three groups: niche (under 100K), mid-range (100K–1M), broad (1M+).
Include 1 branded hashtag placeholder.
```

## Engagement Hooks Library

**Curiosity Hooks**
- "Most people get [topic] completely wrong. Here's why:"
- "I tested [X] for 30 days. The results were unexpected."
- "Nobody talks about this side of [topic]."

**Data Hooks**
- "[X]% of [audience] struggle with [problem]. Here's the fix:"
- "After analyzing [number] [items], I found one pattern."

**Contrarian Hooks**
- "Stop [common advice]. Do this instead."
- "[Popular belief] is a myth. Here's what actually works:"

**Story Hooks**
- "6 months ago I [failure]. Today I [success]. The difference:"
- "My client went from [before] to [after] in [timeframe]."

## Content Repurposing Pipeline

```
One long-form piece → Multiple formats:

Blog post (2000 words)
├── LinkedIn article (condensed, 800 words)
├── 5 LinkedIn text posts (one insight each)
├── Twitter/X thread (key takeaways)
├── Instagram carousel (visual summary)
├── 3 TikTok/Reels scripts (key sections)
├── Email newsletter (intro + link)
└── Quote graphics (5 pull quotes)
```

### AI Repurposing Prompt

```
I have this [blog post / transcript / article]: [paste content]

Repurpose it into:
1. A LinkedIn post (hook + 3 insights + CTA)
2. A Twitter thread (6 tweets)
3. An Instagram carousel outline (6 slides)
4. A 45-second video script

Keep the core message identical. Adapt tone and format per platform.
```

## Quality Checklist Before Publishing

- [ ] Hook is in the first line, no fluff
- [ ] Platform-appropriate length respected
- [ ] Clear CTA (comment, share, follow, save, click)
- [ ] Hashtags researched and tiered (Instagram)
- [ ] Visual matches the copy's tone
- [ ] No jargon the audience won't recognize
- [ ] Spelling and grammar checked
- [ ] Scheduled at optimal posting time for that platform
- [ ] UTM parameters added to any links
