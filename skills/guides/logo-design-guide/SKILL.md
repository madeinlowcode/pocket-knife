# Logo Design Guide

## Core Purpose

A logo is not a brand — it is the anchor of a brand system. A well-designed logo must work at 16px (favicon) and 16 feet (billboard), in color and in black and white, embroidered and on screen. These constraints are not compromises; they are the design brief.

---

## Logo Types

### By Structure
- **Wordmark**: Company name as typography (Google, Coca-Cola, FedEx)
  - Best when name is distinctive and short
  - Typography becomes the brand expression
  - Requires strong typographic execution

- **Lettermark / Monogram**: Initials only (IBM, HBO, GE)
  - Best when full name is long or difficult
  - Requires strong secondary name pairing at launch
  - Highly scalable

- **Icon / Symbol**: Abstract or literal mark (Apple, Nike swoosh, Twitter bird)
  - Requires significant brand exposure to stand alone
  - Never use standalone at launch without name
  - Most flexible for applications

- **Combination mark**: Icon + wordmark together (Airbnb, Mastercard)
  - Most practical for new brands
  - Allows eventual separation of icon and wordmark
  - Clearest communication at all sizes

- **Emblem**: Text inside icon/badge (Harley-Davidson, Starbucks)
  - Conveys heritage, authority, institution
  - Poor scalability — detail lost at small sizes
  - Avoid for digital-first brands

## The Design Brief Before Starting

Answer these before sketching:

1. What industry/sector is this brand in?
2. Who is the primary audience (age, lifestyle, professional context)?
3. What 3 adjectives describe how the brand should feel?
4. Who are the 3 closest competitors? (Design to be distinct from them)
5. Where will the logo primarily appear? (app icon, website, print, apparel, signage)
6. Is there an existing visual asset to build from?

## Scalability as a Design Constraint

### The Scalability Test
Before finalizing any logo, test at these sizes:
- **16px × 16px**: Favicon — only the simplest element survives
- **32px × 32px**: Small icon
- **120px × 40px**: Typical header logo
- **400px × 133px**: Email header
- **2000px × 667px**: Print letterhead
- **Physical**: 1" wide (business card), 12" wide (signage), 3ft wide (banner)

### Reducing Complexity for Small Sizes
Logos often need a simplified version for small contexts:
- **Primary logo**: Full combination mark
- **Compact logo**: Icon only or lettermark
- **Favicon**: Single letter or simplified icon

Create all variants as part of the logo system, not as afterthoughts.

### Detail That Kills Small-Size Reproduction
- Thin strokes under 1pt at small sizes
- Hairline dividers between shapes
- Gradient fills (disappear in embroidery, fax, single-color print)
- Drop shadows or glows
- Complex gradients inside letterforms

## Color Variations

Every logo must work in all of these versions:

### Required Variations
1. **Full color** (primary): The designed version
2. **Single color black**: All elements in pure black
3. **Single color white**: All elements in pure white (for dark backgrounds)
4. **Reversed**: White on brand color background
5. **Grayscale**: For black-and-white printing contexts

### Why This Matters
- Embroidery can only handle solid colors
- Newspaper/fax printing is black only
- Certain sponsorship applications require specific versions
- Legal documents, stamps, and forms often require single-color

### Creating the Single-Color Version
When converting to single color:
- Replace all colored shapes with solid fills
- Maintain visual distinction between elements through shape, not color
- Ensure sufficient contrast between logo and background
- Gradients become solid fills

## Typography in Logo Design

### Font Selection Principles
- Never use a default system font for a professional logo
- Custom letterforms or modified typefaces distinguish from template-feel
- Font must be licensed for commercial use (OFL, commercial license)
- Legibility at small sizes is non-negotiable

### Modifying Letterforms
Even a purchased typeface should be modified:
- Adjust letter spacing (tracking) intentionally
- Modify specific letterforms to create uniqueness
- Create ligatures between connected letters
- Adjust weight, width, or specific strokes

### Typography Pairing in Combination Marks
When icon + wordmark exist together:
- Wordmark: primary identifying typeface
- Tagline: complementary typeface (different weight or family)
- Rule: never use more than 2 typefaces in a logo system

### Kerning
- Optical kerning for all letter pairs — do not rely on automatic kerning
- Space between letters should feel visually equal, not mathematically equal
- "AV", "WA", "To", "LY" are known problem pairs requiring manual adjustment
- Check all letter pairs at multiple sizes

## Negative Space

Negative space in logos creates memorability and secondary meaning.

### Famous Examples of Negative Space
- FedEx arrow (between E and x)
- Amazon smile + A-to-Z arrow
- Carrefour (C formed by negative space)
- Pittsburgh Zoo (tree and gorilla/fish)

### Using Negative Space
- The hidden element should be relevant, not forced
- Negative space works best in lettermarks and simple icons
- Test: can viewers find it without being told? (Optimal)
- Avoid: negative space that looks accidental rather than intentional

## Icon Design Principles

### Geometric Construction
- Construct icons on a grid (8-unit, 24-unit, or 48-unit)
- Use geometric primitives: circles, squares, triangles, golden ratio
- Consistent corner radius across all rounded elements
- Stroke weight consistent or progressively proportional

### Conceptual Strength
Strong icon concepts are:
- **Literal**: Clear depiction of product/service
- **Metaphorical**: Abstract representation of brand value
- **Abstract**: Purely formal, built on brand DNA

Avoid:
- Generic concepts everyone in the industry uses (globe for "global", gear for "settings")
- Overly complex scenes or illustrations
- Literal depictions of cliched concepts

### Optical Corrections
Geometric shapes appear unequal when they are mathematically equal:
- A circle appears smaller than a square of the same width — make circles slightly larger
- Pointed triangles appear lighter than rounded shapes — adjust weight slightly
- A centered element within a circle appears low — move it up 1–2% for optical center

## Logo Construction Rules

### Grid and Geometry
- Design on a grid (typically 100×100 unit artboard)
- Use whole-number or clean-fraction dimensions
- Align all elements to grid anchors
- Consistent spacing unit throughout

### Clear Space
Define a minimum clear space around the logo:
- Typically based on the height of the wordmark's cap height
- Express as: "Minimum clear space = X on all sides"
- This is the enforced buffer between logo and other elements
- Document this in the brand guide

### Minimum Size
Define the minimum reproduction size:
- Wordmark: typically 80–100px wide digital, 1" wide print
- Icon only: typically 16–24px digital, 0.25" print
- Below minimum: use favicon/app icon variant

---

## What Not to Do

- Do not use raster effects (drop shadow, glow, bevel) that can't be vector
- Do not use photos or complex illustrations as logo elements
- Do not use clipart or stock icon libraries unmodified
- Do not create logos that depend on specific background colors to read
- Do not use more than 3 colors in the primary logo
- Do not create logos where the concept relies entirely on color distinction

## File Deliverables Checklist

For a complete logo handoff:

- [ ] Primary full-color logo (SVG + AI/EPS + PNG 2x)
- [ ] Logo on dark background variant (SVG + PNG)
- [ ] Monochrome black version (SVG + PNG)
- [ ] Monochrome white version (SVG + PNG)
- [ ] Horizontal layout variant (if applicable)
- [ ] Icon/mark only variant (SVG + PNG, all sizes)
- [ ] Favicon (16px, 32px, 64px ICO or PNG)
- [ ] App icon (1024px PNG for iOS/Android)
- [ ] All fonts outlined/converted in vector files
- [ ] Color codes documented (HEX, RGB, CMYK, Pantone)
- [ ] Clear space rules documented
- [ ] Minimum size rules documented
- [ ] Usage guidelines (do/don't examples)
